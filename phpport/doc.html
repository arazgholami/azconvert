How to configure:


First you need to have a MySql account. Open lib/config.php file and enter your db account there. The import the db_dump.sql file into your database so the tables and information in them, be created. Now you can use the program.



How to use:


The API is so simple and straightforward. Just get a Convertor subclass (i.e L2AConversion) object, then set its original text, and then call convert(). You can get converted result either by the convert()'s output or by calling the getConvertedResult().


Example:


$convertor = new L2AConversion();

$convertor->setOriginalText(“Lütfən bu yazını ərəb qrafikasına çevir!”);

$result = $convertor->convert(false);


The convert() method gets a boolean variable which indicates if the text would be treated as a Wiki source text or not. If it's true, the convertor engine detects some Wiki formats (like links, templates, interwiki, ...) and takes care of them. Otherwise blindly converts everything.
